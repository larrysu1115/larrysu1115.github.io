#### Quickstart

```bash
$ export FLASK_APP=hello.py
$ flask run
$ python -m flask run
 * Running on http://127.0.0.1:5000/

# serve publicly
$ flask run --host=0.0.0.0 

# enable debug mode - auto reloader
$ export FLASK_DEBUG=1
```
